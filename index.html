INTRODUCTION TO PROJECT

The virus is spreading from person to person through small respiratory droplets. When a person coughs or sneezes, these droplets can also land on nearby surfaces.The number of people being affected by the covid-19 can be visualized using python and matplotlib. In our present situation, knowing the spread of covid-19 is very much important for us . This project helps in giving the statistical analysis of spread of covid-19.This study will focus on drawing insights from publicly available data and statistics on the Coronavirus. The data that will be considered for the study are aggregated COVID-19 patient statistics like daily cases, deaths, recoveries, testing data, etc. Python has been used for carrying out the analysis. With of help of modules like matplotlib and NumPy data visualization can be performed .The data set that are being created will show destruction created by this virus for 100 days . On increase in days the changes being affected by this virus will be seen . At the beginning the virus spread will more due to which the death ratio is also higher . But slowly the death ratio goes down and recovery will at higher rate. All this is shown in data visualization .

SOFTWARE REQUIREMENT

The project code is simulated in VSCODE simulator and Terminal. The libraries used are Matplotlib and NumPy.

Matplotlib:

Matplotlib is a comprehensive library for creating static , animated , and interactive visualizations in Python. This was originally developed by John D. Hunter. Their main aim was to provide best GUI.

Comparison With Matlab

Matplotlib is a Python library that draws graphs that look extremely similar to Matlab. Matplotlib is used in Python which is general purpose programming language. So these can used for developing models .Matplotlib is open-source software which is free .Matlab is numerical computing system that offers an integrated development environment with its own programming language (written in c, c++ and java). Matlab is not open-source software Both are used by Data Scientist or Data analyst for statistical analysis.

NumPy:

NumPy stands for Numerical Python. NumPy is a python library used for working with arrays or list which stores the data .It also has functions for working in domain of linear algebra , fourier transform , and matrices. NumPy was created in 2005 by Travis Oliphnat . It is an open source we can use it freely .
